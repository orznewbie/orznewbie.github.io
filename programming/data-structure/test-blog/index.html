单调栈

单调递增栈是这样一种栈：

每当push一个数a[i]

  1. 如果a[i]比栈顶元素大，则直接入栈
  2. 如果a[i]比较小，则不断pop栈顶元素，直到栈顶元素小于a[i]

单调递增栈有下面3个重要属性：

  1. 栈内元素是递增的
  2. 当元素出栈时，新元素a[i]是出栈元素向后找第一个比其小的元素
  3. 当元素出栈后，新栈顶元素是出栈元素向前找第一个不比其大的元素

单调栈很有用，严格的数学证明暂时不会，直接记住它就行了。
